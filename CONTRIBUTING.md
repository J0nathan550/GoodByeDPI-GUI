
### Build Steps

1. Clone the repository:
```bash
git clone https://github.com/J0nathan550/GoodByeDPI-GUI.git
cd GoodByeDPI-GUI
```

2. Open the solution in Visual Studio:
```bash
GoodByeDPI-GUI.sln
```

3. Build the solution (Ctrl+Shift+B)
